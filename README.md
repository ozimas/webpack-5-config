# Webpack 5 config
### Webpack 5 documentation
[webpack 5 doc](https://webpack.js.org/concepts/)

### Install
install program: [node.js](https://nodejs.org/en/)

### Libraries
Install packages from package.json
```npm
yarn install
```

### Run webpack
run developer application
```npm
yarn dev
```
run build application
```npm
yarn build
```
